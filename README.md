# Diligent_Assignment

how to run:
# 1. Go into your new project's folder
cd my-ecommerce-app

# 2. Install Tailwind and its helper tools
npm install -D tailwindcss postcss autoprefixer

# 3. Create the Tailwind config files (this adds tailwind.config.js and postcss.config.js)
npx tailwindcss init -p

# This command installs the icons
npm install lucide-react

#Run the app-
npm run dev
